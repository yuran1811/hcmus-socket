# Socket Download Server

## Libs

- `socket` - for creating a socket server
- `os` - for file operations
- `threading` - for handling multiple clients
- `json` - data serialization

## Usage

- Run the server

```bash
py server.py
```

- Run the client

```bash
py client.py
```
